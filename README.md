This project like a little habr.

## Available Scripts

Install dependencies:

### `npm run install:back`

### `npm run install:client`

## Available Scripts: 

### Start client

### `npm run start`

Run client without backend

### `npm run start:watch`

Run client and backend

### Start backend

### `npm run server`

Run backend

### `npm run server:watch`

Run backend with nodemon
